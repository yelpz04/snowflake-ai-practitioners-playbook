# Day 9: Snowflake CoWork — From Answers to Action

## Goal
Use Snowflake CoWork as a personal work agent to ask business questions and take action.

## References

## What to Do

2. Ask: "Which region is underperforming in revenue this year?"
3. Ask: "What action should the sales team take for at-risk customers?"
4. Compare the experience to the API-based Cortex Agent from Day 6

- Governed integrations for downstream actions
- Compare: API agent vs. CoWork UI vs. Streamlit app
