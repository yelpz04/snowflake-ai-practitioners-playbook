# Day 12: Cortex Code Data Engineering Skills

## Goal
Use Cortex Code's built-in data engineering skills to create an ingestion + transformation pipeline.

## References
- [Cortex Code governed agent](https://www.snowflake.com/en/blog/cortex-code-governed-agent-data-stack/)

## What to Do
1. Ask Cortex Code: "Create a data pipeline that ingests CSV files from a stage, transforms them, and loads into an analytics table"
2. Observe which built-in skills it uses (pipeline, dynamic tables, etc.)
3. Ask: "Add error handling and logging to this pipeline"
4. Ask: "Optimize this for performance"

## Built-in Skills Available
- Agent creation
- Machine learning workflows
- Data engineering (pipelines, dynamic tables)
- Data governance
- App development
- Performance optimization
