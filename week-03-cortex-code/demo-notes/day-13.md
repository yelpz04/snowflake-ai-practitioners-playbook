# Day 13: Data Governance Skills for Cortex Code

## Goal
Use governance-specific Cortex Code skills to assess governance maturity and fix gaps.

## References
- [Governance skills blog](https://www.snowflake.com/en/blog/engineering/cortex-code-governance-skills/)

## What to Do
1. Ask Cortex Code: "What is my governance maturity score?"
2. Ask: "Scan for sensitive columns missing tags"
3. Ask: "Which tables need masking policies?"
4. Ask: "Generate a governance report for REVENUE_OPS_AI database"
5. Compare results to our custom `governance-checker` skill (Day 14)

## Key Concepts
- Built-in governance skills work via natural language
- Generate and execute governance queries automatically
- Classification, tagging, masking, access control analysis
- Works across databases in the connected account
