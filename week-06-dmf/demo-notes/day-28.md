# Day 28: Snowflake Docs for AI Agents and LLMs

## Goal
Explore how Snowflake docs are now optimized for AI consumption via `llms.txt` and Markdown.

## References
- [Agent-friendly docs](https://docs.snowflake.com/en/release-notes/2026/other/2026-04-15-agent-friendly-docs)

## What to Do
1. Visit `https://docs.snowflake.com/llms.txt` (the hierarchical index)
2. Fetch a docs page in Markdown format (add `.md` to URL)
3. Show how an AI coding agent (Claude Code, Cortex Code) can consume these
4. Discuss: what does "agent-friendly documentation" mean for the industry?

## Key Features
- **llms.txt**: Hierarchical index of all Snowflake docs, optimized for LLM discovery
- **Markdown versions**: Every docs page available as `.md` for cleaner AI consumption
- **Structure**: easier for agents to navigate, search, and cite
