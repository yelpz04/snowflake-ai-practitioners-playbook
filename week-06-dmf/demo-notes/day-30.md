## Recap: What I Built

### 6 POCs, 1 Connected Project
1. **Media + Feedback AI Analyzer** — multimodal AI over images, audio, video
2. **Sales Analyst Agent** — Cortex Agents, semantic views, Intelligence, Artifacts
3. **AI-Governed Project Builder** — Cortex Code skills, DCM, hooks
4. **Governed Multi-Tenant Agent** — identity, guardrails, tenant isolation, ML, migration
5. **Production Monitoring** — QAS, observability, AI telemetry
6. **Production-Ready Assistant** — DMF, anomaly detection, DQ dashboard

### Key Learnings
- Start with exciting demos, not infrastructure
- Semantic views are the bridge between data and AI
- Governance is not optional — it's the production layer
- Multiple AI agents > one universal agent
- DMF belongs after you build, not before
